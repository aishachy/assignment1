১. Typescript এ ইন্টারফেসেস ও টাইপস এর মধ্যে পার্থক্য আলোচনা করো ।

=> 

১। ইন্টারফেস শুধুমাত্র অবজেক্ট টাইপ ডাটার সাথে কাজ করতে পারবে। অবজেক্ট টাইপ বলতে আমরা বুজি যে জাভাস্ক্রিপ্ট এর অবজেক্ট , এরে ও ফাংশন । ইন্টারফেস এর শুরু হয় দ্বিতীয় ব্রাকেট এর মাধ্যমে । এখানে কোনো সমান (=) চিহ্ন ব্যবহার করা হয়না ।
interface Person {
  name: string;
  age: number;
}
অন্যদিকে টাইপ অবজেক্ট টাইপ ডাটার সাথে Primitives, Union, Itersection ও Tuple ডাটার সাথে কাজ করতে পারে। টাইপ এর শুরু হয় সমান (=) চিহ্ন এর মাদ্ধমে। এর অর্থ হচ্ছে যে আমরা টাইপ এর মধ্যে যেকোনো ধরণের ভ্যালু রাখতে পারবো। 
type Person = string | number;

২। ইন্টারফেস এ এক্সটেন্ড এর মাধ্যমে একটি ইন্টারফেস অন্য ইন্টারফেস এর সাথে সংযোগ স্থাপন করতে পারে। ইন্টারফেস এ একই নামের একাধিক ইন্টারফেস লিখলেও ট্যাপেস্ক্রিপ্ট সবগুলো মিলিয়ে একটি একক ইন্টারফেস হিসেবে ব্যবহার করে থাকে।
অন্যদিকে টাইপ এর ক্ষেত্রে এক্সটেন্ড করতে হলে ইন্টারসেকশন ব্যবহার করতে হয়। এবং টাইপ এ একই নাম দিয়ে একাধিক টাইপ লেখা যাবেনা। একাধিক নাম লেখা হলে এটা ভুল দেখাবে। 

৩। টাইপ সাধারণত তখন ব্যবহার করা হয় যখন অবজেক্ট এর মধ্যে বৈশিষ্ট্য কম থাকে। এর মাধ্যমে ইন্টারসেকশন ব্যবহার করে সহজে যেকোনো ধরণের ডাটা বের করে আনতে পারবো। তাছাড়া primitive ডাটা এগুলোর ক্ষেত্রে টাইপ ব্যবহার করা অনেক সহজ। অন্যদিকে ইন্টারফেস সাধারণত তখন ব্যবহার করা হয় যখন অবজেক্ট এর মধ্যে বৈশিষ্ট্য বেশি থাকে। এক্ষেত্রে এক্সটেন্ড ব্যবহার করে সহজে আমরা এক অবজেক্ট এর সাথে অন্য অবজেক্ট এর বৈশিষ্ট্য ব্যবহার করতে পারবো।  এতে আমরা অনেক পরিষ্কার একটা গঠন পাবো। 

৪। এরে ও ফাংশন এর ক্ষেত্রে সাধারণত টাইপ ব্যবহার করলে পরিষ্কার একটা গঠন পাওয়া যাবে যা ইন্টারফেস এর ক্ষেত্রে জটিল গঠন ধারণ করতে পারে।  তবে অবজেক্ট এর ক্ষেত্রে ইন্টারফেস অথবা টাইপ যেকোনো একটাই ব্যবহার করা যেতে পারে। 

২. Typescript এ keyof এর ব্যবহার কি ? একটা উদাহরণ এর মাধ্যমে আলোচনা করো। 

=> 

Typescript এ keyof অপারেটর ব্যবহার করে আমরা কোনো অবজেক্ট টাইপ এর সমস্ত key বের করে আনতে পারবো। অর্থাৎ keyof অপারেটর ব্যবহার করে আমরা অবজেক্ট এর সব key একত্রিত করে একটি টাইপ বানাতে পারবো যা শুধুমাত্র সেই key গুলোর মধ্যে সীমাবদ্দ থাকবে। এর একটি সুবিধা হচ্ছে যে আমরা যদি কোনো ভুল বৈশিষ্ট্য ব্যবহার করি অথবা আমরা যে key ব্যবহার করছি তা যদি টাইপ এর মধ্যে না থাকে তাহলে Typescript আমাদেরকে ত্ৰুটি দেখাবে। এজন্য কোড করা অনেক নিরাপদ হয়। 
নিচে একটি উদাহরণ দেয়া হলো :

type Student = {
     name: string;
     id: number;
     address: string;
     age: number;
     phoneNo: number;
};

function isStudent ( obj: Student, key: keyof Student){
console.log(obj[key]);
}

isStudent ({ name: "Aysha", id: 123, address: "Sylhet", age: 20, phoneNo: 0177777 }, "name"); // কাজ করবে 
isStudent ({ name: "Aysha", id: 123, address: "Sylhet", age: 20, phoneNo: 0177777 }, "email"); // কাজ করবেনা কারণ এখানে email টি Student অবজেক্ট এর মধ্যে নাই।
